### 1.1 Translators 更新流程

1. 将新增翻译器放到 Zotero 数据存储目录下的 `translator` 目录中
2. 关闭并重启 Zotero
3. 浏览器插件 -> 右健 -> 拓展选项 -> Advanced -> Update Translator
4. 刷新要抓取的信息源（网页）
5. 抓取
